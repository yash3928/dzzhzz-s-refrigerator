냉장고 관리 프로그램 v0.3 standalone

실행 방법
1. 압축을 풉니다.
2. index.html을 더블클릭해 브라우저에서 실행합니다.

주요 기능
- 홈 / 식재료 / 등록 / 설정 화면
- 모바일 하단 메뉴
- 냉장 / 냉동 / 실온 현황
- 유통기한 임박(3일 이내) / 경과 필터
- 등록 / 수정 / 삭제 / 검색
- JSON 백업 내보내기 / 가져오기
- Firebase 미연결, 브라우저 LocalStorage 저장

데이터 호환
- 기존 저장 키 fridge-manager-items-v2를 그대로 사용합니다.
- 같은 브라우저, 같은 실행 주소에서 기존 v0.2 데이터가 유지됩니다.
